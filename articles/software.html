Software I Recommend

Updated January 9, 2024

For Everyone:

Firefox: Basically every other browser is controlled by google.

Charter Font: Gives you a nice serif look while doing above the bare minimum with your typography.

For Computer People:

Thunderbird: Collecting all your email and RSS into a real offline program feels like a captain's chair for the internet.

Micro: Everybody needs a text editor that works in the terminal. I like this one because it works like any other modern program like the text input in Firefox or thunderbird. The others like Vim and Nano have these legacy systems of keybindings and operation. If you want a normal terminal text editor, use Micro.

Arch Linux: I got a computer genius friend who hyped me into Arch when I was dithcing Window 11 instead of like Mint or something. I would like to thank him for that because just by trying to use this thing I have turned into a low level baby IT guy. Now that I am proficient, I like the under engineering type philosophy this thing is made with.

Sway Window Manager: Most interfaces feel like they have indigestion compared to Sway. My iPhone especially feels slow and floaty.

Gnome/GNU: A person using something as out there as a tiling window manager should probably have some tight, unique boutique program for everything. But when I need a tool, I know Richard Stallman and his boys have me covered down to the principles. I like the Gnome file explorer, disk tools, text editor (gedit), and font tools especially.
