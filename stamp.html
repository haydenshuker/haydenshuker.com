stamp

Stamp is the tool I make and use to update my website.

Its a python script that uses way to many libraries, has big floppy functions, and breaks in ways that mess up your files system. It is the third version of my static site generator that I've been using since summer 2024.

features:

  • attaches headers and footers to every page
  • writes a blog roll
  • updates a backlog list and a rss feed with each article post
  • updates changes to git
