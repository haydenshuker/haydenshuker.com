static site generator

Websites are strange because the whole site is like a cohesive digital thing, but every page wants to be its own text file, so editing the whole can be laborious as if it were an actual physical object.

I write the content of a page by hand, the raw HTML. The templates like the header and footer are saturated in by my site generator so I can change every file at once. For anyone with a bit of coding a recommend making one as an easy project.
